Release notes for Kendo UI v2012.2.913

Core

What's new

  • Allow functions to be used in data attributes during declarative initialization

DataSource

What's new

  • Add request end event

What's fixed

  • DataSource sync does not properly accepts new models
  • Operation descriptors should not be passed to the transport when client operation mode is enabled
  • cancelChanges does not converted nested objects to ObservableObject
  • Values are not converted if external model definition is specified
  • Requests are blocked after custom error is processed during read operation.
  • Nested object are not observable after empty response of CUD operation is received
  • Null values for fields of type string are converted to "null" when remote binding is used

Globalization

What's new

  • Support for escaping a percent sign in a custom numeric format.

HierarchicalDataSource

What's new

  • Sorting and filtering expressions are applied when loading child datasources

What's fixed

  • get method does not find items in child datasources
  • Loading nodes with hasChildren=false queries the child datasource

MVVM

What's new

  • kendo.init recognizes javascript reference as the role data attribute value
  • role data attribute accepts full namespace as widget identifier

What's fixed

  • Parent field is not updated if it is changed from a nested template
  • kendoPager does not support source binding
  • Old event handlers are invoked when the binding has changed
  • Incorrect action type is passed through the change event of the parent object in case item is added to nested array

Validator

What's new

  • Validator template should not encode the message text

What's fixed

  • Disabled input element are validated

Web

What's new

  • Introduce DropTargetArea widget - manage live drop targets within an area

What's fixed

  • DropTarget drop event is not triggered in IE8 when element inside iframe
  • Multiple kendo.init kendo.bind calls do not unbind previous event handlers
  • FX gets disabled if DataViz is included after Web
  • JavaScript error is thrown when the editor value is bound via MVVM
  • The checked binding does not work as expected with checkbox whose value attribute is set
  • The "value" binding does not work if set before the "source" binding

AutoComplete

What's fixed

  • Widget does not update the popup's position after rebind
  • Dropdowns should resize according to opening element with percentage width
  • Popup is opened after the widget loses focus

Calendar

What's fixed

  • Widget does not work in mobile device

ComboBox

What's new

  • Select an item by index/value when populate the datasource of the widget

What's fixed

  • Widget does not encode the text of the option element.
  • Widget does not update the popup's position after rebind
  • Cannot enter a custom value if the list of items is re-generated
  • Widget does not open popup element on search
  • Dropdowns should resize according to opening element with percentage width
  • Popup is opened after the widget loses focus
  • Widget does not select the correct item of the child combobox on onload
  • When text property is set and widget is not bound, the change event is raised on blur

DatePicker

What's new

  • Copy class names of the input element to the widget's wrapper

What's fixed

  • Widget does not update the input value when set a valid value with diff format.
  • Widget doesn't work in mobile browser with native HTML5 input support
  • Widget navigates to the next month when the last day of the month is selected.
  • Cannot choose date in IE with browser mode 9 and document mode lower than 9
  • Widget does not work in mobile device

DateTimePicker

What's new

  • Copy class names of the input element to the widget's wrapper

What's fixed

  • Widget does not update the input value when set a valid value with diff format.
  • Widget doesn't work in mobile browser with native HTML5 input support
  • Widget navigates to the next month when the last day of the month is selected.
  • Widget uses an incorrect time format when the options are set with setOptions method
  • Cannot choose date in IE with browser mode 9 and document mode lower than 9
  • Widget does not work in mobile device

DropDownList

What's new

  • Add focused state style to the DropDownList
  • Select an item by index/value when populate the datasource of the widget

What's fixed

  • Widget does not encode the text of the option element.
  • Widget does not update the popup's position after rebind
  • Dropdowns should resize according to opening element with percentage width

Editor

What's fixed

  • Editor nested in window may not work in some cases
  • JavaScript error is thrown in IE8 when pressing enter in an ordered list
  • Editor content is lost after appending content to document.body
  • Executing inline format commands may cause loss of data in IE7/8
  • Editor iframe causes security warning in IE with HTTPS protocol
  • Shift+Enter does not scroll to caret

Grid

What's new

  • Add support for preventing refresh through cancelling the DataBinding event
  • Grid should not display the string "null" or "undefined" when the cell value is null or undefined

What's fixed

  • Filtering on foreign key column which contains literal symbol throws an error
  • Custom validation rules are not executed when Popup edit form template is used
  • Selection does not work with jQuery 1.8.1
  • Cursor is not persisted during column resizing
  • Data is in incorrect order when grouped on a field with same date value and sorted by another.
  • Fix Grid horizontal scrollbar not visible in IE8 in virtual scroll scenarios
  • Footer row cannot be scrolled horizontally after grid is re-bound
  • Clicking cancel in a grid nested in another grid's popup closes the parent popup
  • PageSizes DropDownList is hidden when Grid is populated through the MVVM using a DataSource
  • The new record is not removed after clicking the cancel button of the popup editor
  • Column menu fails to initialize if field of a complex object is specified
  • Error is thrown when setting more then two filters on the same field and filtering is enabled
  • Footer has a scrollbar gap in mobile devices.
  • Column reordering doesn't work properly on command columns
  • Error is thrown when edit form is shown for complex objects in popup and in-line mode
  • Detail template cannot be set as configuration option through MVVM
  • Column menu doesn't work properly when reorder columns
  • Footer is not shown on initial load if autoBind is false
  • IE throws an error when Grid is bound through MVVM and column is sorted if filtering is enabled
  • Popup windows remain in DOM during grid editing

ListView

What's new

  • Add support for preventing refresh through cancelling the DataBinding event

What's fixed

  • ListView refresh throws error in IE8

Menu

What's fixed

  • closeOnClick disables openOnClick temporarily.
  • Widget doesn't work in Chrome for Android
  • openOnClick not reset after closeOnClick

NumericTextBox

What's new

  • Add focus() method to NumericTextBox

PanelBar

What's fixed

  • Clicking on .k-link elements within panelbar content collapses root items

Splitter

What's fixed

  • jQuery UI Draggable does not work in splitter panes
  • Iframes in panes may trap events and restrict resizing

TabStrip

What's fixed

  • Adding dynamic items does not work on iPad
  • Remove method doesn't remove all contents if more than one
  • Refresh method doesn't remove all tabs
  • Removing tabs not functioning properly on touch devices.

TimePicker

What's new

  • Copy class names of the input element to the widget's wrapper

What's fixed

  • Widget doesn't work in mobile browser with native HTML5 input support

TreeView

What's new

  • Allow template to be set through data-template attribute
  • Introduce simplified checkbox configuration and checkChildren configuration option
  • Introduce setDataSource method that allows dynamic changing of dataSource
  • Checkboxes set the dataItem checked state
  • Introduce dataBound event
  • Improve performance with big data sets
  • Allow nodes to be reloaded by setting the Node loaded flag
  • Initially expanded load-on-demand items are loaded through the datasource
  • text(node, 'text') now updates the text of the given node

What's fixed

  • Reloading of subnodes leaves subgroup in faulty state
  • Dropping items over unfetched nodes throws JS error
  • append() method does not work with unfetched nodes
  • Disabled items show wrong toggle arrow indicator
  • Initialization from pre-populated Node hierarchy does not render subnodes
  • Widget doesn't work in Chrome for Android
  • Drag&Drop hover states not removed on touch devices
  • dataXFields do not work with complex getters ("Product.Name")
  • Popup-based widgets are broken when in TreeView template
  • Calling append() with kendo.observable argument throws a JS error
  • Calling `expand()` on load-on-demand nodes has no effect
  • sourceNode event argument in dragEnd event is outside the DOM
  • Updating text field in HierarchicalDataSource does not change node text

Upload

What's fixed

  • Selecting multiple files in Safari/Windows results in failed uploads

Window

What's new

  • Error event passes error details to event handlers
  • Refresh event is triggered when refreshing static windows
  • Allow refresh() method to render iframe when passing iframe: true

What's fixed

  • Minimized windows can be resized after they are moved
  • Initialization always triggers refresh event
  • Inherited window widgets throw a JS error when calling close() or destroy()
  • Make the Window trigger a refresh event when an iframe is used
